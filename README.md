# Twitch Panel Generator (Twitchパネル作成ツール)

このツールは、Twitchのプロフィール欄に飾る「パネル画像」を、ブラウザ上で誰でも簡単に作れるツールです。
お手持ちの画像を用意するだけで、Photoshopなどの難しい画像編集ソフトを使わずに、ブラウザ上で簡単に文字と合成したオリジナルパネルを作成できます。

## 🌟 特徴
- **完全プライバシー保護**: あなたが選んだ画像やフォントはサーバーに一切保存されません。ブラウザを閉じれば完全に消去されるため、機密情報が含まれた画像でも安心して使えます。
- **1ファイルで動く**: 難しい設定は不要です。ファイルを1つサーバーに置くだけで動きます。
- **ヘッダ機能**: 見出し用の画像を上部にくっつけることができます。

## 🚀 使い方

### 1. 準備（サーバーへの設置）
1. お使いのサーバー（さくらのレンタルサーバなど）に `twitch_panel_generator.php` をアップロードします。
2. ブラウザ（ChromeやSafariなど）で、アップロードしたファイルのURLにアクセスします。
   （例： `https://あなたのサイト/twitch_panel_generator.php`）

### 2. パネルの作り方
画面の左側が「設定」、右側が「プレビュー（完成予定図）」になっています。

1. **背景画像を選ぶ**
   - 「画像をブラウザに追加」のボタンを押して、パソコンやスマホから背景にしたい画像を選びます。
2. **（オプション）ヘッダ画像を付ける**
   - パネルの上部に見出し用の画像を付けたい場合は、「ヘッダ画像を有効にする」にチェックを入れます。
   - 同じように「画像をブラウザに追加」から見出し用の画像を選びます。
   - その上に文字を入れることもできます！
3. **文字（テキスト）を入力する**
   - 「テキスト」の枠の中に、パネルに書きたい文章を入力します。
   - 文字の大きさ、色、配置（左寄せ・中央揃えなど）、縁取りなどを自由に変更できます。
4. **好きなフォント（字体）を選ぶ**
   - 最初から用意されているフォントのほかに、「カスタムフォントをブラウザに追加」から、お好きなフォントファイル（.ttf など）を追加して使うこともできます。
5. **プレビューで確認する**
   - 「プレビュー更新」ボタンを押すと、右側に完成予定の画像が表示されます。
   - 納得がいくまで、設定を変えてはプレビューを押して調整してください。
6. **画像を保存する！**
   - 完成したら「保存ファイル名」に好きな名前（英語がおすすめ）を入力し、「PNG保存」ボタンを押します。
   - パソコンに完成した画像がダウンロードされます！あとはTwitchにアップロードするだけです。

## ⚠️ 注意事項
- このツールは、アップロードされた画像やフォントをサーバー内に一切保存せず、処理完了後に即座に破棄する仕組みになっています。
- そのため、**ブラウザの画面を閉じたり、再読み込み（F5キーなど）をすると、追加した画像やフォントはすべてリセットされて消えてしまいます。**
- 作業が途中の場合は、必ず「PNG保存」をして画像をダウンロードしておいてください。
